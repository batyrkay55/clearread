Настройки чтения

На чём основан дизайн

  • Интервалы важнее шрифта. Специальные «дислексик-шрифты» не показали устойчивого преимущества перед обычным чистым sans-serif — разница в межбуквенном и межстрочном интервале работает надёжнее.
  • Длина строки. Рекомендуемая длина строки — 60–70 символов, текст выровнен по левому краю без выравнивания по ширине.
  • Межстрочный интервал 1.5–2× снижает визуальную скученность текста.
  • Не белый фон. Тёплый неяркий оттенок вместо чистого белого снижает контрастную усталость при чтении.
  • Размер шрифта имеет значение сам по себе — увеличенный кегль ускоряет чтение независимо от гарнитуры.
Прототип для учебного проекта. Упрощение текста выполняется языковой моделью — итоговый текст стоит перепроверять на смысловую точность перед использованием.
